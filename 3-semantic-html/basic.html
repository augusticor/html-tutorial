VIM editor

Vim is a clon with additions of "VI" the editor made for Unix OS, this was released to public in 1991. Vim is designed for use both from a CLI(command line interface) as and a standalone application in a graphic user interface.

Vim's interface is not based on menus or icons but on commands given in a text user interface.
Vim has 12 modes, here i mention the basic ones:
  • Normal
  • Visual
  • Select
  • Insert
  • Command Line
  • Ex

Vim has customization, more features, scripts and much more but you can read about this in the web ...
